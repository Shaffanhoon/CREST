Academic/Professional Qualifications

CREST Registered Penetration Tester

Recognised by CESG as providing the minimum standard for CHECK Team Member status. This exam is designed to assess a candidate’s ability to carry out basic vulnerability assessment and penetration testing tasks. The exam is in two parts, a practical assessment and a multiple choice section aimed at assessing the candidate’s technical knowledge. For more information click the link please Registered Penetration Tester.

CSTP Ethical Web Application Hacking

Provider: 7Safe. This 2-day web application ethical hacking course covers the fundamentals of the industry-recognised OWASP Top Ten. You will have access to a functional ASP.NET and PHP application through which theory is reinforced by way of practical exercises. The course demonstrates hacking techniques - there’s no better way to understand attacks than by doing them yourself - but this is always done with defence in mind and countermeasures are taught throughout. For more information click the link please CSTP Ethical Web Application Hacking.
